# GET /albums/#{id}/mute
***
`/albums/#{id}/mute`

### Description
***
Check if a user has muted an album


### Parameters
***

|header| description| type |required? |default|
|:---------|:--------------|:----------:|:------------:|:------------:|
|**id**| the id of the album to share|integer|x||
|**parameter**| **description**| **type** |**required?** |**default**|


### Response
***
200 if success
